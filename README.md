# intangible_website
# 薪火拾遗 · 非遗中国互动网站
> 面向中华非物质文化遗产的前端互动展示平台

An interactive website for intangible cultural heritage. It introduces four Chinese heritage items: Yuxian paper-cut, Suzhou embroidery, Peking Opera facial masks and Jingdezhen porcelain. The website includes interactive features such as 3D particle tree, heritage puzzle game, knowledge quiz, fragment collection, blind box draw and message board, to promote traditional Chinese intangible cultural heritage.

该网站介绍河北蔚县剪纸、苏州苏绣、京剧脸谱、景德镇制瓷四项非遗；集成 3D 非遗粒子树、非遗拼图闯关、非遗知识答题、非遗碎片收集与盲盒抽取、守护留言板等互动功能，借助网页形式推广中华非物质文化遗产。

## 📖 项目介绍
本项目为非遗文化主题静态网页，围绕四项经典非遗项目展开展示：
河北蔚县剪纸、苏州苏绣、京剧脸谱、景德镇制瓷。

网站集成多种互动功能：
✅ 3D非遗粒子树可视化展示
✅ 非遗拼图闯关小游戏
✅ 非遗知识问答系统
✅ 非遗碎片收集 & 非遗IP盲盒抽取
✅ 守护留言板，记录大众对非遗传承寄语

采用原生 HTML + CSS + JavaScript 开发，无需后端，开箱直接运行。

## 💡 项目意义
借助现代化网页交互形式，生动展示传统非遗魅力，拉近年轻人与非物质文化遗产的距离，助力传统文化传播。
